Engine: WAVE API od WebAIM. Profesionální evaluace po aplikaci CSS a JavaScriptu. 100 testů zdarma, pak ~$0,025/test.

Nastavení WAVE API klíče klíč nenastaven

WAVE API vyžaduje vlastní klíč. Registrace zdarma s 100 kredity na wave.webaim.org/api/register. Po přihlášení najdete klíč v účtu na wave.webaim.org/api/login. Klíč se uloží do localStorage tohoto prohlížeče a nikam jinam neodejde.

Zobrazený jako hvězdičky. Pro kontrolu klikněte na "Zobrazit".
Pokročilé: proxy URL (kvůli CORS) přímé volání

WAVE API obvykle nepodporuje CORS pro klientské volání — pokud se Vám stránka v prohlížeči zastaví na CORS chybě, nasaďte si vlastní proxy worker (vzorový 10řádkový Cloudflare Worker je v patičce). Worker WAVE odpověď přepošle a doplní CORS hlavičku Access-Control-Allow-Origin: *. Klíč ani URL se ve workeru neukládají — jsou v query parametrech volání.

Aplikace bude místo https://wave.webaim.org/api/request volat Vaši proxy se stejnými query parametry. Pokud chcete vlastní placeholder, použijte {params}, jinak se parametry doplní samy.

Test stojí 3 kredity (reporttype=4: plný detail s CSS selektory a contrast daty). Vrátí se Vám i počet zbývajících kreditů.

Pohání WAVE API 3.1 od WebAIM (Utah State University). WAVE provádí evaluaci po aplikaci CSS a JavaScriptu.

Vzor Cloudflare Worker proxy (nutné, pokud volání selže na CORS)
  1. Zaregistrujte si free Cloudflare účet a nainstalujte wrangler.
  2. Vytvořte projekt: wrangler init wave-proxy → "Hello World" Worker.
  3. Obsah src/index.js nahraďte tímto:
export default {
  async fetch(request) {
    const incoming = new URL(request.url);
    const target = new URL('https://wave.webaim.org/api/request');
    incoming.searchParams.forEach((v, k) => target.searchParams.set(k, v));
    const upstream = await fetch(target.toString(), {
      headers: { 'User-Agent': 'WAVE-proxy/1.0' }
    });
    const body = await upstream.text();
    return new Response(body, {
      status: upstream.status,
      headers: {
        'Access-Control-Allow-Origin': '*',
        'Content-Type': upstream.headers.get('content-type') || 'application/json;charset=utf-8'
      }
    });
  }
};
  1. Deploy: wrangler deploy. Dostanete URL typu https://wave-proxy.váš-účet.workers.dev/.
  2. Tu vložte do "Pokročilé" výše. Aplikace nyní volá WAVE přes Vaši proxy.
